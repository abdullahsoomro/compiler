CLR(1) Algorithm

Initialise
// Setup stack = [0] // push start state a = next_token() // read first token
Push state 0 onto the stack. Read the first token from input into the lookahead register a.
Main Loop
loop: s = top(stack) // peek state act = ACTION[s, a] if act == shift t: push(a, t) // push symbol + state a = next_token() elif act == reduce r: β = RHS(r) // production rhs pop |β| symbols A = LHS(r) // production lhs t = top(stack) push(A, GOTO[t, A]) elif act == accept: return SUCCESS else: return ERROR
Key Distinctions vs SLR
SLR reduces A→α when a ∈ FOLLOW(A) — globally.
CLR(1) reduces only when a equals the item's exact lookahead, computed from the specific derivation path. This eliminates false conflicts.
