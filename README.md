# compiler
This repository contains complete notes on parsing techniques in compiler construction, including LR(0), SLR(1), LALR(1), and Canonical LR(1). It provides step-by-step derivations of item sets, closure and goto operations, parsing tables, and explains shift-reduce conflicts with clear examples for deep understanding and exam preparation.

## Parsing in Compiler Design (LR, SLR, LALR, CLR)

A complete, step-by-step guide to **bottom-up parsing techniques** in compiler construction.  
This repository focuses on **LR-family parsers** with detailed explanations, canonical item construction, parsing tables, and conflict analysis.
